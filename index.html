Java Game Application: 'Memory Game'

AY 2022-2023

On our second semester of the the Academic Year 2022-2023, we were tasked to create a basic Java Game Application in which we were divided into groups where each group contains five members. Our project is a "Memory Game" in which users would have to press a series of buttons. The buttons contains numbers from 1 to 12. As the user progresses, each level becomes more and more difficult in which the user would have to remember more and more numbers. These numbers would first appear for the user to memorize, and if he presses the first number, the rest of the numbers will disappear. It's up to the user then to recall which number should he press next.

The application makes heavy use of Java programming language. As such, I was one of the members who were tasked to do the back-end programming of the application. I also contributed to the design and styling of the application.

  • Tasked on the back-end programming of the application
  • Contributed to the application design/style
